# IntroduceMySelf

`IntroduceMySelf`는 자기소개 내용을 바탕으로 만든 인터랙티브 퀴즈 사이트입니다.
교육실습 및 수업 예시 자료로 제작되었으며, 학생들이 기계학습의 개념과 유형을 학습한 뒤 `Codex`를 활용해 직접 자기소개 퀴즈 사이트를 만들어보는 실습에 앞서 참고할 수 있는 예시 사이트입니다.

## 제작 목적

이 사이트는 단순한 자기소개 페이지가 아니라, 학생들이 다음과 같은 흐름으로 수업을 이해하도록 돕기 위한 예시입니다.

- 기계학습의 개념과 활용 사례를 학습한다.
- 지도학습, 비지도학습, 강화학습 등 기계학습의 주요 유형을 구분해본다.
- 생성형 AI 도구인 `Codex`를 활용해 웹사이트를 구현하는 과정을 경험한다.
- 자신의 정보를 바탕으로 퀴즈 형태의 자기소개 사이트를 직접 설계하고 제작해본다.

## 수업 속 활용 맥락

수업 시간에는 먼저 기계학습과 기계학습 유형에 대해 학습한 뒤, AI 코딩 도구를 활용하여 결과물을 만드는 실습으로 확장할 수 있습니다.
이 사이트는 그 과정에서 학생들에게 보여주는 예시 결과물로, “어떤 형태의 산출물을 만들 수 있는지”를 직관적으로 안내하는 역할을 합니다.

특히 학생들은 이 예시를 통해 다음과 같은 아이디어를 얻을 수 있습니다.

- 자기소개를 단순한 글이 아닌 퀴즈 형식으로 구성할 수 있다.
- 사용자의 참여를 유도하는 인터랙티브 요소를 넣을 수 있다.
- 웹페이지에 게임 요소를 더해 더 재미있는 결과물을 만들 수 있다.

## 주요 구성

- 객관식 자기소개 퀴즈
- 정답 여부를 알려주는 피드백 모달
- 간단한 메모리 게임 보너스 스테이지
- 밝고 친근한 분위기의 단일 페이지 웹사이트 구성

## 기대 효과

이 예시 사이트를 통해 학생들은 AI를 활용한 코딩이 자신의 아이디어를 디지털 결과물로 구현하는 도구가 될 수 있음을 경험할 수 있습니다.
또한 학습한 내용을 바탕으로 직접 주제를 정하고, 구조를 설계하고, 결과물을 개선하는 과정을 통해 창의적 문제 해결 경험을 쌓을 수 있습니다.

## 실행 방법

정적 웹페이지이므로 별도의 설치 없이 `index.html` 파일을 브라우저에서 열면 실행할 수 있습니다.

## 파일 구성

- `index.html`: 사이트의 기본 구조
- `styles.css`: 화면 스타일 및 레이아웃
- `script.js`: 퀴즈와 메모리 게임 동작 로직
